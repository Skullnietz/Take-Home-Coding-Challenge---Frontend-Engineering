# Advantec • User Directory (Take Home)

**Listar / ver / crear / editar** perfiles con React + Vite + TypeScript. Mock API con MirageJS.

## Scripts
- `npm run dev` — desarrollo (Vite)
- `npm run build` — build de producción
- `npm run preview` — servidor de preview
- `npm run test` — tests con Vitest
- `npm run lint` — ESLint

## Arranque
```bash
npm install
npm run dev
```

## Endpoints mockeados (MirageJS)
- `GET /profiles`
- `GET /profile/:id`
- `POST /profile`
- `PATCH /profile/:id`
